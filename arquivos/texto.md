A interface File provê informações sobre arquivos e permite ao JavaScript  a acessar seu conteúdo.

São geralmente recuperados a partir de um objeto [FileList](https://www.letras.mus.br/jao/amor-pirata/) que é retornado como resultado da seleção, pelo usuário, de arquivos através do elemento [<input>](https://www.letras.mus.br/jao/santo/), a partir do objeto [DataTransfer](https://www.letras.mus.br/jao/doce/) utilizado em operações de arrastar e soltar, ou a partir da API `mozGetAsFile()` em um [HTMLCanvasElement](https://www.letras.mus.br/gracie-abrams/thats-so-true/). Em Gecko, códigos com privilégiios podem criar objetos File representando qualquer arquivo local sem a intereção do usuário (veja [Implementation notes](https://www.letras.mus.br/gracie-abrams/i-know-it-wont-work/) para mais informações.).
